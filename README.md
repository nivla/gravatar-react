# React Gravatar
React component for rending gravatar images.

## Install
```
npm install gravatar-react
```

## Usage
Simple example
```
<Gravatar email="dickson.alvin@gmail.com" />
```
or
```
<Gravatar
  email="dickson.alvin@gmail.com"
  size=100
  rating="PG"
  alt="Alvin Dickson profile"
  secure
```
